# Package index

## Core Pipeline

The three-step workflow: specify a design, add study conditions, and run
the simulation.

- [`irt_design()`](https://sward1.github.io/irtsim/reference/irt_design.md)
  : Create an IRT Design Specification
- [`irt_study()`](https://sward1.github.io/irtsim/reference/irt_study.md)
  : Define Study Conditions for an IRT Simulation
- [`irt_simulate()`](https://sward1.github.io/irtsim/reference/irt_simulate.md)
  : Run an IRT Monte Carlo Simulation

## Results Analysis

Summarize, visualize, and extract recommendations from simulation
results.

- [`summary(`*`<irt_results>`*`)`](https://sward1.github.io/irtsim/reference/summary.irt_results.md)
  : Summarize IRT Simulation Results
- [`plot(`*`<irt_results>`*`)`](https://sward1.github.io/irtsim/reference/plot.irt_results.md)
  : Plot IRT Simulation Results
- [`plot(`*`<summary_irt_results>`*`)`](https://sward1.github.io/irtsim/reference/plot.summary_irt_results.md)
  : Plot Summary of IRT Simulation Results
- [`recommended_n()`](https://sward1.github.io/irtsim/reference/recommended_n.md)
  : Find the Minimum Sample Size Meeting a Criterion Threshold

## Utilities

Helper functions for parameter generation and planning.

- [`irt_params_2pl()`](https://sward1.github.io/irtsim/reference/irt_params_2pl.md)
  : Generate 2PL Item Parameters
- [`irt_params_grm()`](https://sward1.github.io/irtsim/reference/irt_params_grm.md)
  : Generate GRM Item Parameters
- [`irt_iterations()`](https://sward1.github.io/irtsim/reference/irt_iterations.md)
  : Compute Required Monte Carlo Replications
